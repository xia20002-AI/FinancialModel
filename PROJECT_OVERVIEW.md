## 📊 核心功能

### 1. 三表财务模型 ✅
- **损益表** (Income Statement)
  - 收入预测 (5年可配置)
  - 成本结构 (COGS/OpEx)
  - 利润计算

- **资产负债表** (Balance Sheet)
  - 流动资产 (现金、AR、库存)
  - 固定资产 (PPE滚存)
  - 负债与权益

- **现金流量表** (Cash Flow)
  - 经营现金流
  - 投资现金流
  - 融资现金流

- **自动对账**
  - 资产 = 负债 + 权益
  - 现金流对账
  - 留存收益验证

### 2. DCF估值分析 ✅
- WACC计算 (使用CAPM模型)
- 自由现金流预测
- 终端价值计算
- 企业价值 → 股权价值 → 每股价值
- 敏感性分析 (WACC vs 终端增长率)

### 3. 数据可视化 ✅
- 收入/利润趋势图
- 利润率分析
- 现金流趋势
- 企业价值构成
- 敏感性分析热力图

## 🚀 快速开始 (3步)

### 最快方式：Docker
```bash
cd financial-analysis-platform
docker-compose up -d
```

**完成！** 立即访问：http://localhost:3000

### 本地开发 (3个终端)
```bash
# 终端1: Python财务引擎
cd finance-engine
pip install -r requirements.txt
python api_server.py

# 终端2: Express后端
cd backend
npm install
npm run dev

# 终端3: React前端
cd frontend
npm install
npm start
```

## 💻 使用流程

输入股票代码 (如 MSFT) 或公司名称，系统自动显示匹配结果

设置财务参数：

- 📊 成本比例 (COGS/OpEx/CapEx)

### 第3步：运行分析
一键执行：
- ⬇️ 自动获取历史财务数据

- 💎 执行DCF估值
- 📊 生成可视化

### 第4步：查看结果
- **📈 财务报表标签页** - 三表详细数据 + KPI
- **💰 DCF估值标签页** - 估值结果 + 投资建议
- **📊 可视化标签页** - 交互式图表
